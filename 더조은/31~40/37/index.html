position

독립적인 위치를 설정할 수 있다

  • position이 적용된 요소는 좌표를 지정할 수 있다
    left, right, top, bottom 좌표축을 사용한다
    left와 right중 택일, top, bottom중 택일
  • position(abs, fix)이 적용된 요소들은 기존 디스플레이 속성이 사라진다
  • position이 적용되면 z-index속성을 가지게 된다
  • absoulte : 문서의 position이 적용된 상위요소 기준
  • fixed : 웹브라우저 기준
  • relative : 자식 absolute요소의 좌표 기준점으로 활용,
    기본 display 속성을 유지함
    좌표를 지정하기 전 위치를 0점으로 계산

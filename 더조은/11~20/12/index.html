리스트 요소

유사기능 요소들을 그룹화 하는 목적으로 사용

ul

순서가 없는 목록
type 속성 : disc, circle, square

예시) 내 취미

  • 노래
  • 게임
  • 운동
  • 배드민턴

ol

순서가 존재하는 목록
type 속성 : 1, A, a, I, i

예시) 내 취미 비용순위

  1. 게임
  2. 운동
  3. 배드민턴
  4. 노래

dl

용어정리목록

예시) web언어 설명

html
웹문서 마크업 언어
css
돔구조 스타일 디자인 언어

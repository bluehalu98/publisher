table 제작

  • table(block레벨과 유사), tr(table rows), td(cell)
  • 모든 줄의 cell의 갯수는 동일해야 한다
  • fieldset의 넓이는 동일하다
  • table에 넓이가 적용되었을 경우 모든 td에 넓이(px)를 지정하지 않는다
  • 모든 td에 넓이(px)가 적용되었을 경우 table에는 넓이를 지정하지 않는다
  • table에는 높이를 지정하지 말것
  • td의 vertical-align속성은 cell안의 글자 수직 위치를 결정
  • cell의 병합
    - colspan, rowspan
    - 병합된 cell에는 크기를 지정하지 말것

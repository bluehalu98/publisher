form 요소

사용자가 입력, 변경할 수 있는 모든 요소

  • 데이터의 전송 범위를 form요소(block레벨)로 설정한다
  • fieldset : form의 정보범위
  • legend : fieldset의 제목
  • label : 폼요소에 대한 참조를 제공(웹접근성에 대한 요소)
  • input
    • 직접입력 type
      • text (size, maxlength, placeholder)
      • password (size, maxlength, placeholder)
      • number (placeholder, min, max)
      • email (size, maxlength, placeholder)
      • tel (size, maxlength, placeholder)
    • 선택 type
      • radio (name:라디오버튼일 경우에만 기능이 달라짐, checked)
      • checkbox (checked)
      • date
      • file(accept)
    • 버튼 type
      • submit : 전송기능
      • reset : 폼요소 초기화
      • button : 아무기능없는 버튼
  • select, option
    option 객채만 label이란 안내속성을 사용할수 있음
  • textarea
    태그안의 여백을 생성하지 말것, cols : 가로넓이 , rows : 세로넓이

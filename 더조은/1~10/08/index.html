이미지출력하기

경로

  • 절대경로
    - 루트경로부터 시작하는 방식 - 도메인 절대경로 : http 시작하는 도메인 경로
    - 서버 절대경로 : /로 시작하는 방식
  • 상대경로
    - 작성하는 문서 기준으로 부터 위치를 찾는 방식 - 현재폴더 : ./ (작성하는 문서 기준의 현재 폴더) - 상위폴더 : ../ (작성하는 문서 기준의 상위 폴더)

img 요소

  • 웹접근성을 준수하기 위해서 alt(대체문자)속성을 추가해야 함
  • heading요소 안에는 img요소만 존재해서는 안됨
  • Ratio(비율)을 유지하기 위해서 넓이나 높이중 하나만 설정
  • Ratio이 변할경우 object-fit속성과 같이 사용

background 속성

심미성(시각적 의미)을 위해 사용하길 권장

배경의 반복, 위치

  • 배경이미지는 패턴형태로 출력된다
  • background-repeat속성 (repeat, no-repeat, repeat-x, repeat-y)
  • background-position속성 (가로 : left center right, 세로 : top center bottom)

배경색과 배경이미지를 같이 사용하기

배경이미지 크기

  • baackground-size
    cover : 객체를 빈틈없이 배경으로 채운다
    contain : 객체 안에 모든 배경이 적용되기

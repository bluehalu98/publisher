CSS 개념

cascading style sheet의 약자로 웹문서의 디자인을 처리하는 언어임
cascading -> 폭포수, 우선순위가 있다

특징과 주의사항

  • 우선순위(cascading)가 존재함
    인라인스타일 > ID 선택자 > 클래스 선택자 > 자손 > 부모
  • 상속성(inherit)을 가지고 있음(상속이 되는 속성과 되지 않는 속성이 존재함)
  • head요소안에서 style요소를 선언후 선택자{속성:값} 형태로 작성한다

선택자(selector)

  • 태그 선택자(type selector)
    태그명{속성:속성값; 속성:속성값; 속성:속성값}
  • 클래스 선택자(class selector)
    .클래스이름{속성:속성값;}방법으로 선언후 html요소에서 class속성으로 호출
    반복호출(재사용)이 가능하고 수정관리가 용이함
  • ID 선택자(ID selector)
    #ID이름{속성:속성값;}방법으로 선언후 html요소에서 ID속성으로 호출
    동일한 ID는 사용할 수 없음
  • inline style
    html요소의 style속성을 추가해서 한줄로 속성과 값을 설정
    수정관리가 매우 비효율적임, 자바스크립트를 이용해 css를 변경할 경우 사용됨

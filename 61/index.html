sass (css전처리기)

  • 자바스크립트파일(scss)로 작성된 css문법을 css파일로 파싱(변경, 해석)하는 형태
  • 사전에 개발 환경을 구축해야 함
  • 작성은 scss, 연결은 css
  • 선택자의 접근이 간단해진다
  • &선택자 : 부모선택자 (&를 포함한 중괄호의 선택자를 의미)
  • sass의 독특한 문법
    · 선택자 중괄호 안에서 미디어쿼리를 작성가능
    · rgba(green,0.8)
  • 일반 css 변수 사용 예시 
            
        :root{--main-color:red;}
        body{color:var(--main-color);}
    scss 변수사용 예시 
            
          $main-color:rgb(41,71,110);
          body{color: $main-color;}
  • scss의 변수나 mixin은 별도의 파일로 제작후 import를 이용해 탬플릿형태로 구조화
  • !주의 : sass의 변수나 mixin은 수정할 경우, 해당 변수나 mixin을 사용하는 모든 css파일을 재 컴파일링(새로저장) 해야한다
    !팁 : css의 변수형태를 사용하길 권장
  • 문자사이에서 변수를 출력하고 싶은 경우
    예시 
            $size:10;
        body{
          font-size:#{$size}px ; font-size:10px
        }
  • @mixin, @include
    목적 : 반복적이고 복잡한 css의 수정관리가 용이
    인자 : 파라미터를 이용하여 다양한 값을 제어가 가능하다 
              - 예시
          @mixin center-width($w,$max-w){
            width:$w;
            max-width:$max-w;
            margin:0 auto;
          }
          div{
            @include center-width(95%, 1200px)
          }
  • 반복문
    • for구문(through:까지, to:전까지) 
                  @for $i from 1 through 6{
              &:nth-child(#{$i}){
                animation-delay: #{($i-1)*0.2}s;
              }
            }
    • each구문, 객체와 사용할때 유용 
                  span{
              $icons:(
                1 : '\f573',
                2 : '\f095',
                3 : '\f1ac',
                4 : '\f279'
              );
              @each $n , $icon in $icons{
                &:nth-child(#{$n}){
                  @include icon-font(before,$icon)
                }
              }
            }

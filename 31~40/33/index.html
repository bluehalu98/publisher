1. flex박스의 개념

  • 부모컨테이너에게 flex를 적용
  • flex컨테이너는 block레벨의 성격을 갖는다
  • flex컨테이너의 자식객체들은 기본 display속성이 아니다

2. flex-direction

  • row : 기본값, 그리드가 수평배치
  • column : 그리드가 수직배치
  • row-reverse : 그리드가 오른쪽 역순(순서가 바뀐다)
  • column-reverse : 그리드가 아래쪽 역순(순서가 바뀐다)

3. justify-content

  • flex 컨테이너에 적용, flex컨테이너 자식들의 위치분할
  • flex-start : 기본값
    flex-direction이 row일 경우 flex-start 왼쪽
    flex-direction이 column일 경우 flex-start 위쪽
    flex-direction이 row-reverse일 경우 flex-start 오른쪽 역순
    flex-direction이 column-reverse일 경우 flex-start 아래쪽 역순
  • space-between : 부모컨테이너의 공간을 자식들이 분배해 갖는다
    flex-direction이 row일 경우 좌우 분할
    flex-direction이 column일 경우 상하 분할
    flex-direction이 row-reverse일 경우 좌우 분할
    flex-direction이 column-reverse일 경우 상하 분할
  • flex-end : 반대쪽으로 자식들을 정렬한다
    flex-direction이 row일 경우 오른쪽으로 정렬
    flex-direction이 column일 경우 아래로 정렬
    flex-direction이 row-reverse일 경우 왼쪽으로 역순으로 정렬
    flex-direction이 column-reverse일 경우 위쪽으로 역순으로 정렬
    순서는 reverse일경우에만 역순이다

4. align-items

  • :strecth
    flex-direction이 row일 경우 자식들의 높이가 부모의 높이와 같아진다
    flex-direction이 column일 경우 자식들의 넓이가 부모의 넓이와 같아진다
  • :center
    flex-direction이 row일 경우 자식들이 원래 높이가 되며 부모의 수직중앙으로 정렬
    flex-direction이 column일 경우 자식들이 원래 넓이가 되며 부모의 수평중앙으로 정렬
  • :flex-start
    flex-direction이 row일 경우 자식들이 원래 높이가 되며, 왼쪽 정렬
    flex-direction이 column일 경우 자식들이 원래 높이가 되며 윗쪽 정렬
  • :flex-end
    flex-direction이 row일 경우 자식들이 원래 높이가 되며, 오른쪽 정렬
    flex-direction이 column일 경우 자식들이 원래 높이가 되며, 아래쪽 정렬
  • justify-content와 align-items속성 모두 center로 설정하면 객체의 정중앙으로 이동한다

5. flex 속성

    flex-direction이 col인경우 자식들이 높이를 나누어 갖는다

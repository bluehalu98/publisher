float

정확하고 다양한 레이아웃을 구현

  • float이 적용되면 디스플레이 레벨 성격이 사라진다
    block레벨의 경우 기본 넓이가 사라짐, margin의 auto값 적용 안됨
    inline레벨의 경우 크기나 padding도 적용할 수 있음
  • clear fix :
    float이 적용된 자식객체의 높이를 부모컨테이너가 인식할 수 있도록
    css 가상엘리먼트를 설정하는 방법
  • float객체의 다음객체(next sibiling)는 반드시 float와 clear중 하나를 선택해야 함
  • float이 적용된 요소들만 margin-top이 적용됨

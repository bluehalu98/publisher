1. CSS 동적제어

  • css({'css속성':'값', 'css속성':'값'})
    요소의 인라인 스타일을 제어하는 방식
    toggle기능 구현이 안됨
  • addClass('클래스명1 클래스명2'), removeClass('클래스명'), toggleClass('클래스명')
    여러개의 클래스를 추가할 경우 동일한 속성이 중복되지 않도록 css를 설계
  • show(), hide(), toggle()
    inline스타일의 display속성을 변경하는 방식
  • fadeIn(), fadeOut(), fadeToggle()
    inline스타일을 이용해서 투명해지면서 사라지거나 나타나는 방식
    마지막 명령어를 바로 실행시키려면 stop()명령어를 먼저 적용
  • slideUp(), slideDown(), slideToggle()
    inline스타일을 이용해서 아코디언 방식
    마지막 명령어를 바로 실행시키려면 stop()명령어를 먼저 적용

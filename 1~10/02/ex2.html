여러가지 단축키 및 이론

명령어 및 기능

  1. alert() : 가로안의 내용을 알림창으로 알려준다

Ctrl 단축키

  • Ctrl + B : 프로젝트 관리창 열고 닫기
  • Ctrl + k + O : 폴더 열기
  • Ctrl + D : 같은 단어 찾기
  • Ctrl + W : 켜져있는 파일 닫기
  • Ctrl + 방향키 : 띄어쓰기가 되어있는 부분을 스킵하며 이동

Shift 단축키

  • Shift + Alt + 방향키 : 아래나 위로 선택한 줄의 내용을 복사
  • Shift + Alt + 클릭 : 아래 한줄로 멀티커서 활성화
  • Shift + Alt + A : 드래그한 부분의 주석화
  • Shift + Tab : 선택한 줄의 들여쓰기 한칸씩 취소
  • Shift + Alt + F : 파일 전체를 표준 들여쓰기로 변경

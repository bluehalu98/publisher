anchor 요소

hyper link를 구현하는 목적으로 사용(button과 다르게 페이지가 바뀔때만 anchor를 사용한다)

외부링크, external link

  • hyper link를 적용할 목적으로 사용
  • hyper link를 적용하려면 href(hyper reference)속성을 추가
  • target으로 브라우저 윈도우를 제어(_self(디폴트값이며 현재창을 링크로 이동), _blank(새창을 링크로 이동))
  • anchor객체는 기본적으로 agent style이 적용된다
네이버로 이동하기
구글로 이동하기
다음페이지로 이동

내부링크(책갈피), internal link

  • 위치와 이동버튼을 설계
  • 위치표식은 name속성, 이동버튼 href속성을 사용
  • href속성의 #(임시링크)은 현재 페이지를 의미함
